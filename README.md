# Project CRUD USER VUE FLASK

## BACKEND

### Enter the backend folder
```
cd backend/
```

### Install pipenv
```
pip install pipenv
```

### Start environment
```
pipenv shell
```

### Install dependencies
```
pipenv install -r requirements.txt
```

### Start backend
```
python3 run.py 
```

## FRONTEND

### Enter the frontend folder
```
cd frontend/
```

### Install dependencies
```
npm i
```

### Start frontend
```
npm run serve
```
